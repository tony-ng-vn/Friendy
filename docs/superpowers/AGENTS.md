# Superpowers Docs Agent Instructions

This directory stores Superpowers planning artifacts.

- Specs live in `docs/superpowers/specs/`.
- Implementation plans live in `docs/superpowers/plans/`.
- Keep specs about decisions and constraints; keep plans about executable steps.
- Do not bury current repo navigation here. Put cross-repo navigation in `REFERENCE.md`.
- If a plan is executed, preserve it and update `implementation-notes.html` with deviations, tradeoffs, and verification.
