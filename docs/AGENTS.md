# Docs Agent Instructions

Use this directory for durable product, planning, handoff, and goal context.

- Keep docs navigational and decision-oriented.
- Prefer linking to the source file, spec, or plan instead of copying large blocks across documents.
- When implementation changes the architecture, update `implementation-notes.html` and the relevant doc in the same change.
- Keep `docs/goals/` focused on goal-mode execution files and `docs/superpowers/` focused on Superpowers specs/plans.
