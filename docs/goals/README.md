# Friendy Goal Queue

Use these long-form goal files one at a time. In Codex, start a new goal by pointing `/goal` at the file instead of pasting the whole prompt.

Before writing or editing goals, read `goal-writing-guide.md`.

Recommended order:

1. `contextual-memory-capture-v2-goal.md`
2. `relationship-agent-response-composer-goal.md`
3. `contact-event-verification-queue-goal.md`

Each goal should keep `docs/goals/PLAN.md`, `docs/goals/EXPERIMENTS.md`, and `docs/goals/EXPERIMENT_NOTES.md` updated while running.

For exact copy/paste prompts, use `friendy-goal-runbook.md`.
